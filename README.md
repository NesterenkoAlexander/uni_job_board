# Центр Карьеры БГИТУ

Платформа для размещения и поиска вакансий для студентов и выпускников БГИТУ.

## Быстрый старт

```bash
git clone <репозиторий>
cd uni-job-board
cp .env.example .env
npm install
npm run seed
npm run dev
```

Сервер запустится на `http://localhost:3000`.

## Настройка

Отредактируйте `.env` при необходимости:

| Переменная | Описание | По умолчанию |
|---|---|---|
| `PORT` | Порт сервера | `3000` |
| `SESSION_SECRET` | Секрет для сессий | `random_secret` |
| `ADMIN_USERNAME` | Логин администратора | `admin` |
| `ADMIN_PASSWORD_HASH` | bcrypt-хеш пароля | хеш `admin123` |

### Смена пароля администратора

Сгенерируйте новый хеш:

```bash
node -e "const b=require('bcryptjs');console.log(b.hashSync('ваш_пароль',10))"
```

Вставьте результат в `ADMIN_PASSWORD_HASH` в `.env` и выполните `npm run seed` (если админ уже существует — пересоздайте БД, удалив `database.sqlite`).

## Использование

- **Публичная часть:** `http://localhost:3000` — просмотр и поиск вакансий.
- **Админ-панель:** `http://localhost:3000/admin/login` — управление вакансиями.
